Portfólio Expositivo

Este projeto consiste em um portfólio Web desenvolvido com o objetivo de apresentar os conceitos que aprendi, e evolução na área de Desenvolvimento de Sistemas.

A aplicação foi construída utilizando HTML e CSS. O projeto busca aplicar conceitos fundamentais de Desenvolvimento Front-end, como organização estrutural, estilização hodierna e experiência do usuário.

Objetivos do projeto:

- Consolidar conhecimentos em Desenvolvimento Web;
- Desenvolver uma interface voltada à apresentação profissional;
- Servir como vitrine para projetos e habilidades em desenvolvimento.

Acesso ao projeto

• https://guilhermejaraujo15.github.io/PortfolioExpositivo/

Considerações finais:

Este portfólio representa minha evolução prática como desenvolvedor, reunindo conhecimentos adquiridos ao longo dos estudos e projetos realizados. O projeto continuará sendo atualizado com novas funcionalidades, melhorias visuais e inclusão de projetos mais completos.
